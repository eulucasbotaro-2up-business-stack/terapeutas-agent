# Calculadora de ROI — Agente IA para Terapeutas

## Formula Basica

```
ROI = (Economia mensal - Investimento) / Investimento x 100
```

---

## Cenario Conservador (Plano Essencial R$197/mes)

| Metrica | Valor |
|---|---|
| Tempo gasto respondendo WhatsApp | 1.5h/dia |
| Valor da hora do terapeuta | R$150 |
| Custo diario do tempo perdido | R$225 |
| Custo mensal (20 dias uteis) | R$4.500 |
| IA resolve 50% das mensagens | -R$2.250 economizados |
| Investimento mensal | R$197 |
| **Economia liquida** | **R$2.053/mes** |
| **ROI** | **1.042%** |

---

## Cenario Medio (Plano Profissional R$297/mes)

| Metrica | Valor |
|---|---|
| Tempo gasto respondendo WhatsApp | 2h/dia |

| Custo diario do tempo perdido | R$400 |
| Custo mensal (20 dias uteis) | R$8.000 |
| IA resolve 60% das mensagens | -R$4.800 economizados |

| Receita adicional (3 x R$200) | +R$600/mes |
| Investimento mensal | R$297 |
| **Economia + receita extra** | **R$5.103/mes** |
| **ROI** | **1.718%** |

## Argumentos-chave para a venda

1. **Payback imediato:** O investimento se paga no primeiro dia de uso
2. **Tempo livre:** 1-2 horas extras por dia para atender mais pacientes
3. **Pacientes retidos:** Resposta instantanea = menos desistencias
4. **Escalabilidade:** Atenda 2x mais pacientes sem contratar ninguem
5. **Custo comparativo:** Menos que 1 consulta por mes
