- **Subtítulos:** Inter, weight 600 (semibold)
- **Corpo:** Inter, weight 400-500 (regular/medium)
- **Código/técnico:** JetBrains Mono ou Fira Code

## ESPECIFICAÇÕES TÉCNICAS

### Exportação de Imagens
- Web: WebP ou PNG, máx 150KB por imagem de suporte
- Impressão: PDF vetorial, mínimo 300dpi
- OG image (landing): 1200×630px

### Mapa Natal
- Dimensão: 1200×900px (figura Joel) / 900×900px (tradicional)
- DPI: 150 (suficiente para WhatsApp)
- Formato: PNG com fundo branco
- Tamanho alvo: < 300KB por imagem
